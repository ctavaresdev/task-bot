# 🤖 Task Bot

Meu projeto acadêmico desenvolvido para faculdade com o objetivo de criar um bot de organização de estudos para Discord utilizando Python.

## 🎓 Sobre o projeto

O Task Bot aplica conceitos de:
- Python
- Discord.py
- JSON
- Automação e organização acadêmica

## 📚 Funcionalidades

- Adicionar, listar e remover tarefas
- Gerenciar matérias
- Agendar provas
- Sistema de ajuda com comandos

## ⚙️ Comandos principais

!tarefa  
!tarefas  
!remover_tarefa  
!materia  
!materias  
!remover_materia  
!prova  
!provas  
!remover_prova  
!comandos

## 🔒 Segurança

O token é protegido via arquivo `.env`.

## 📦 Instalação

pip install -r requirements.txt  
python bot.py

## 🔗 Adicionar ao Discord

Discord Developer Portal → OAuth2 → URL Generator → bot
